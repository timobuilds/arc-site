# Applied Research Club

Site for [Applied Research Club](https://arc-site.vercel.app) — centaur logo and block-letter text on black.

## Quick start

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build

```bash
npm run build
```

Output in `dist/`. Preview with `npm run preview`.

## Deploy to Vercel

1. Push to GitHub
2. Import the repo in [Vercel](https://vercel.com)
3. Vercel auto-detects Vite — no build config needed
4. Deploy

Or use the CLI:

```bash
npm i -g vercel
vercel
```
